# Type War ⚡

Type War is a minimal typing speed test web app that measures your **WPM (Words Per Minute), accuracy, and errors** in real time.

## 🚀 Features

- Real-time typing test
- WPM calculation
- Accuracy tracking
- Error highlighting
- Animated title effect
- Clean minimal UI
- Result modal with blur background

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- GSAP (animation)

## 📊 Metrics Calculated

- **WPM (Words Per Minute)**
- **Accuracy %**
- **Error count**
- **Top speed**

## 🎮 How It Works

1. Generate random words.
2. Start typing.
3. The app tracks:
   - Correct characters
   - Mistakes
   - Time taken
4. When the text ends, the result card appears.


Built as a frontend project to practice **JavaScript logic and DOM manipulation**.
